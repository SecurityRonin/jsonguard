Privacy Policy

Project: jsonguard
Maintainer: Security Ronin Ltd
Effective date: 2026-05-22

1. No Data Collection

jsonguard is a Rust library that runs entirely within your own process. It does not transmit, log, or store any data. It has no network access, no telemetry, and no analytics. All processing happens locally on the machine where your code runs.

2. No Dependencies That Collect Data

jsonguard has zero production dependencies. The only optional dev-dependency (proptest) is used solely during testing and is never included in your compiled binary.

3. This Website

These documentation pages are hosted on GitHub Pages. GitHub may collect standard server-side access logs (IP address, browser user-agent, referring URL) as described in the GitHub Privacy Statement. Security Ronin Ltd does not have access to those logs and does not set any cookies or run any tracking scripts on this site.
